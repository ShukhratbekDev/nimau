# HTTP nima?

HTTP (**Hypertext Transfer Protocol**) - bu web-brauzerlar va serverlar o'rtasida ma'lumotlarni uzatish uchun ishlatiladigan protokol. U internet orqali veb-sahifalarni, rasmlarni, videolarni va boshqa resurslarni yuklashga imkon beradi.

## Asosiy Xususiyatlari

- **Stateless (Holatsiz)**: Har bir HTTP so'rovi mustaqil bo'lib, oldingi so'rovlar haqida ma'lumot saqlanmaydi.
- **Request-response modeli (So'rov-javob modeli)**: Mijoz (brauzer) so'rov yuboradi, server esa ushbu so'rovga javob beradi.
- **URI (Uniform Resource Identifier)**: Resurslarni aniqlash uchun ishlatiladi, masalan, URL (Uniform Resource Locator).

<Note>
    HTTPS - HTTP protokolining kengaytirilgan versiyasi bo'lib, ma'lumotlarni shifrlash orqali xavfsiz ulanishni ta'minlaydi.
</Note>
