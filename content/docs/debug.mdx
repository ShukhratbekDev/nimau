**Debug** (Tuzatish) — bu ilovalarida xatoliklarni topish, tuzatish va bu jarayonni kuzatish uchun ishlatiladigan jarayon.

## Asosiy Afzalliklar
- **Xatolarni aniqlash**: Dasturdagi xatolarni tez topish imkonini beradi.
- **Jarayonni kuzatish**: Ilova jarayonini kuzatish va funksiyalarni tekshirish imkoniyati.
- **Diagnostika**: Xatolarni tahlil qilish va ilovani barqaror qilish.

## Foydalanish

<Callout>
  **Debug** — Ilovalarda xatoliklarni topish, tuzatish va bu jarayonni kuzatish uchun muhim jarayon hisoblanadi.
</Callout>

### Misol

Node.js ilovasini debug qilish uchun `--inspect` bayrog'idan foydalanish misoli:

```js
// debugExample.js
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello, Debugging World!');
});

app.listen(3000, () => {
  console.log('Server is running on port 3000');
});
```

```bash
node --inspect-brk=9229 debugExample.js
```
