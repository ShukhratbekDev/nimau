### Misol

Oddiy ob'ekt yaratish va unga kirish misoli:

```js
const person = {
  name: 'NimaU',
  age: 25,
  greet: function () {
    console.log('Salom, ' + this.name);
  },
};

console.log(person.name); // => NimaU
console.log(person.age); // => 25
person.greet(); // => Salom, NimaU
```

Bu misolda `person` ob'ekti `name`, `age` xossalar(properties)ga va `greet` metodga ega.
