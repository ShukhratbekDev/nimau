## Asosiy Afzalliklar

- **Qayta ishlatish**: Funksiyalarni bir necha bor qayta ishlatish mumkin.
- **Modullash**: Katta kodlarni kichik, boshqarilishi oson bloklarga bo'lish.
- **Parametrlar**: Funksiyalar funktsional parametrlar qabul qilish va natija qaytarish imkoniyatiga ega.

## Foydalanish

<Callout>**Functions** JavaScriptda qayta ishlatiladigan kod bloklarini yaratishga imkon beradi.</Callout>

### Misol

Oddiy funksiyalarni yaratish va chaqirish misoli:

```js
function greet(name) {
  return `Salom, ${name}!`;
}

console.log(greet('NimaU')); // => Salom, NimaU!
```

Funktsiyalar arguments va return orqali qiymatlar bilan ishlay oladi.
