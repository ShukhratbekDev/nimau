### Misol

Oddiy massiv yaratish va uni qayta ishlash misoli:

```js
const numbers = [1, 2, 3, 4, 5];

console.log(numbers[0]); // => 1
numbers.push(6);
console.log(numbers); // => [1, 2, 3, 4, 5, 6]
numbers.pop();
console.log(numbers); // => [1, 2, 3, 4, 5]
```

Bu misolda `numbers` massiviga qiymatlar qo'shiladi va o'chiriladi.
