**Variables** (O'zgaruvchilar) â€” bu JavaScriptda malumotlarni saqlash uchun foydalaniladigan xotira joylari hisoblanadi.

## Asosiy Afzalliklar

- **Malumot saqlash**: Malumotlarni vaqtincha saqlash imkoniyati.
- **Nomi va qiymati**: Har bir o'zgaruvchi o'z nomini va qiymatini saqlaydi.
- **Dinamik tip**: O'zgaruvchilar turli malumot turlarini saqlay oladi.

## Foydalanish

<Callout>
  **Variables** JavaScriptda malumotlarni saqlash uchun ishlatiladi va ular bir necha xil malumot turlarini qabul
  qilishi mumkin: String, Number, Boolean, Object, Array, va boshqalar.
</Callout>

### Misol

O'zgaruvchilarni e'lon qilish va ulardan foydalanish misoli:

```js
let name = 'John';
const age = 30;
var isMarried = false;

console.log(name); // => John
console.log(age); // => 30
console.log(isMarried); // => false
```
