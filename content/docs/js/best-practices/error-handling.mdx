## Asosiy Afzalliklar

- **Barqarorlik**: Xatolarni to'g'ri boshqarish ilovaning barqarorligini ta'minlaydi.
- **Foydalanuvchi tajribasi**: Xatolar to'g'ri ko'rsatilishi foydalanuvchi tajribasini yaxshilaydi.
- **Diagnostic**: Xatolarni aniqlash va ularni bartaraf etish imkonini beradi.

### Misol

Oddiy xato boshqarish misoli:

```js
try {
  let result = riskyOperation();
} catch (error) {
  console.error('Xato aniqlandi:', error);
} finally {
  console.log('Xato boshqarish jarayoni tugadi.');
}
```

Bu misolda, `try` blokidagi kodda xato yuz bersa, `catch` bloki ishlaydi va xatoni konsolga chiqaradi. `finally` bloki esa har doim ishlaydi.
