**Express.js** (Express) — bu Node.js uchun eng mashhur va keng qo'llaniladigan web framework bo'lib, u web va mobil(API qismini) ilovalarni yaratishni soddalashtiradi.

## Asosiy Afzalliklar
- **Yengillik**: Soddalashtirilgan va intuitiv interfeys.
- **Moslashuvchanlik**: HTML sahifalar, API-lar va boshqa ko'pgina funktsiyalarni yaratish imkoniyati.
- **Keng qo'llab-quvvatlash**: Katta hamjamiyat va ko'plab plaginlar.

## Foydalanish

### Misol

Bir oddiy Express.js serveri yaratish misoli:

```js
const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, World!');
});

app.listen(port, () => {
  console.log(`Server is running at http://localhost:${port}`);
});
```
