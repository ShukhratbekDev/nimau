## Asosiy Afzalliklar
- **Paketlarni boshqarish**: Paketlarni o'rnatish, yangilash va boshqarish imkoniyati.
- **Moslashuvchanlik**: Keng dasturiy paketlar katalogi.
- **Ishlab chiqarish**: Paketlarni ishlab chiqarishda samaradorlikni oshirish.

## Foydalanish

<Callout>
**npm** dasturchilarga Node.js uchun keng dasturiy paketlar katalogiga ega bo'lib, ularni o'rnatish va boshqarish imkonini beradi.
</Callout>

### Misol

npm yordamida loyiha yaratish va paket o'rnatish misoli:

```bash
# Loyiha yaratish
npm init -y

# Express.js paketini o'rnatish
npm install express
```
