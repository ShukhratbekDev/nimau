**HTTP** (Hypertext Transfer Protocol) - bu web-brauzerlar va serverlar o'rtasida ma'lumotlarni uzatish uchun ishlatiladigan protokol. U internet orqali veb-sahifalarni, rasmlarni, videolarni va boshqa resurslarni yuklashga imkon beradi. HTTP serverdan mijozga (brauzer) ma'lumotlar uzatishni ta'minlaydi.

## HTTP protokolining asosiy xususiyatlari

- **Stateless (Holatsiz):** Har bir HTTP so'rovi mustaqil bo'lib, oldingi so'rovlar haqida ma'lumot saqlanmaydi.
- **Request-response modeli:** Mijoz (brauzer) so'rov yuboradi, server esa ushbu so'rovga javob beradi.
- **URI (Uniform Resource Identifier):** Bu resurslarni aniqlash uchun ishlatiladi, masalan, URL (Uniform Resource Locator).

**HTTPS** Haqida ma'lumotlar
**HTTPS** - bu **HTTP** protokolining xavfsiz versiyasi bo'lib, S harfi "Secure" (Xavfsiz) degan ma'noni anglatadi. HTTPS orqali amalga oshiriladigan ma'lumot uzatish jarayonida ma'lumotlar shifrlangan bo'ladi. Bu esa internet orqali uzatilayotgan ma'lumotlarning uchinchi tomonlar tomonidan o'qilishini oldini oladi. Shu sababli, bank xizmatlari yoki boshqa shaxsiy ma'lumotlarni ishlatadigan veb-saytlar odatda **HTTPS** orqali ishlaydi.
